# MAS-IP

MAS-IP is a auto ip Changer tool you can use this in termux.

## Installation


```bash
pkg install git -y && git clone https://github.com/mazbha1019/MAS-IP.git && cd MAS-IP && bash install.sh
```

## Usage

Just Type : masip
